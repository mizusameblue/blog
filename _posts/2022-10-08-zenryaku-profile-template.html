--- title: 前略プロフィール風テンプレートを作ってみた tags: - Web 制作 ---

このブログにプロフィールページでも設けようと思ったが、どうせ作るなら普通じゃつまらない。

そこでひらめいたのが、今は亡き前略プロフィールの再現だった。

「前略プロフィール　テンプレート」で検索しても、それ風の HTML と CSS を配布しているところは見つからなかったので、自分で1から作ることにした。

テンプレート作成方法

サービスが終了したサイトといえどウェブアーカイブにはデータが残っている。

本当はソースコードを丸パクリしようと思ったが、やたらと div まみれだし style 属性に CSS を直に埋め込んでるしで、煩雑な印象だ。

そのため、デザインを元に極力近付けつつ HTML と CSS を書き直した。

プロフィールの項目は以下の知恵袋に載っていたものを流用。

  • 前略プロフィールの全項目を教えてください! - HNHNの由来性別誕生日... - Yahoo!知恵袋

ただ、下の記事の画像を見る限り、他にも項目はあるっぽい。

  • 今さらだけど「前略プロフィール」をやってみた - Togetter
